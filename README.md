# SynSemClass @ LREC 2026

Code and data accompanying the paper accepted at **LREC 2026**:

**Automatic Suggestions Help Extending Eventive Ontology: A Case Study on SynSemClass**

---

## 📌 Overview

This repository contains the implementation and experiments supporting our work on improving and extending the **SynSemClass eventive ontology** using automatic suggestion methods.

The project explores how machine-assisted recommendations can support ontology extension with improved efficiency and consistency.

---

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/strakova/SynSemClassLREC2026.git
cd SynSemClassLREC2026
```

2. Create a virtual environment

```
python -m venv venv
```

3. Install dependencies

```
pip install -r requirements.txt
```

The exact pinned package versions are provided in:

```
requirements_freeze.txt
```
